## 📋 Prerequisites

Before starting, ensure you have:

- [ ] **Python 3.10+** installed — [Download from python.org](https://www.python.org/downloads/)
- [ ] **A code editor** — VS Code, PyCharm, or any editor of your choice
- [ ] **Basic command line knowledge** — Navigating directories, running commands
- [ ] **Git** (recommended) — For version control exercises

## 💡 Learning Tips

### Active Recall
After each chapter, close the tab and write down what you remember. Then re-open and check.

### Hands-On Practice
**Don't just read** — type every example yourself:
```bash
python -c "print('Hello, Python learner!')"
```

### Spaced Repetition
- Review Week 1 concepts at the end of Week 2
- Review Weeks 1-2 concepts at the end of Week 3
- Take all 6 practice tests in the final week

### Build Projects
- Week 1: Calculator, number guessing game
- Week 2: To-do list CLI, file organizer
- Week 3: REST API wrapper, web scraper
- Week 4: Flask blog, data analysis dashboard
